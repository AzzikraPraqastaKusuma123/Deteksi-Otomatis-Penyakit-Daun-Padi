# SQL Database Changes for `agricultural_resources` table

To enhance the `agricultural_resources` table with rich, AI-generated explanations, the following columns were added. These columns are designed to store detailed information generated by the Google Gemini API, allowing for structured content similar to the disease details.

---

### `ALTER TABLE` Statement to Add New Columns:

```sql
ALTER TABLE `agricultural_resources`
ADD COLUMN `gemini_overview_id` TEXT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci DEFAULT NULL AFTER `description`,
ADD COLUMN `gemini_overview_en` TEXT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci DEFAULT NULL AFTER `gemini_overview_id`,
ADD COLUMN `gemini_usage_tips_id` TEXT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci DEFAULT NULL AFTER `gemini_overview_en`,
ADD COLUMN `gemini_usage_tips_en` TEXT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci DEFAULT NULL AFTER `gemini_usage_tips_id`,
ADD COLUMN `gemini_benefits_json` JSON DEFAULT NULL AFTER `gemini_usage_tips_en`,
ADD COLUMN `gemini_rekomendasi_tambahan_json` JSON DEFAULT NULL AFTER `gemini_benefits_json`;
```

---

### Explanation of New Columns:

1.  **`gemini_overview_id` (TEXT)**
    *   **Purpose:** Stores a concise general overview or introduction about the agricultural resource in **Indonesian**, generated by the Gemini AI.
2.  **`gemini_overview_en` (TEXT)**
    *   **Purpose:** Stores a concise general overview or introduction about the agricultural resource in **English**, generated by the Gemini AI.
3.  **`gemini_usage_tips_id` (TEXT)**
    *   **Purpose:** Stores practical usage tips, dosage, or application instructions for the resource in **Indonesian**, generated by the Gemini AI.
4.  **`gemini_usage_tips_en` (TEXT)**
    *   **Purpose:** Stores practical usage tips, dosage, or application instructions for the resource in **English**, generated by the Gemini AI.
5.  **`gemini_benefits_json` (JSON)**
    *   **Purpose:** Stores a structured list of the key benefits or features of the resource. It is stored as a JSON array where each item is an object (e.g., `[{"point": "Benefit 1"}, {"point": "Benefit 2"}]`). This field is designed to be language-agnostic.
6.  **`gemini_rekomendasi_tambahan_json` (JSON)**
    *   **Purpose:** Stores additional recommendations, such as good combinations with other products or optimal usage conditions. It is stored as a JSON array of objects (e.g., `[{"recommendation": "Additional tip 1"}]`). This field is also language-agnostic.

These additions allow your application to store and display more comprehensive and structured details for each agricultural resource, leveraging the capabilities of the Google Gemini API.
